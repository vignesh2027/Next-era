✨ Overview

NEXT ERA is a futuristic high-end 3D web experience crafted to deliver cinematic visuals, immersive interactions, ultra-smooth animations, and modern UI/UX principles.

This project focuses on:

Advanced visual storytelling

Smooth GPU-accelerated animations

Interactive futuristic UI

Modern responsive layouts

Luxury-level user experience

Performance-first frontend engineering


Designed fully in [Figma](https://www.figma.com/?utm_source=chatgpt.com) and developed with modern web technologies.
